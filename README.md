# 📝 To-Do List App (Python)

## 📌 Overview

A simple command-line To-Do application to manage daily tasks.

* Add, view, and delete tasks
* Data saved using file handling

## 🛠️ Tech Stack

* Python

## ▶️ Run

```bash
python todo.py
```

* Functions & user input
* Basic project structure
